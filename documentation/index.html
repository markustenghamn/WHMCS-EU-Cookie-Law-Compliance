Installation

Once you have downloaded your files find the zip file called anveto_cookies.zip and extract it. You should now have a folder called "anveto_cookies" with two .php files inside. These need to be uploaded to the addons directory inside your WHMCS installation. You can use any FTP program to do this. Simply upload the folder along with the two files inside of it to the /modules/addons directory. Once uploaded you should see this addon inside your WHMCS installation.

In order to check if the install was successfull open your web browser and navigate to the administration area of your WHMCS installation. Then, using the menus, navigate to Setup -> Addon Modules and look at the list of addons that show on the page. You should see the addon called "Cookie Law Compliance" and if it isn't already activated click the activate button. The cookie law compliance message will now show on your WHMCS installation.

Configuration

In order to configure the addon navigate to Setup -> Addon Modules in your WHMCS administration area. Here you will see an addon called "Cookie Law Compliance". To the right click the "Configure" button in order to bring up the available options. Here you may edit the text using regular HTML code by changing the value in the "Display text" field. You can also change the background, text and link colors by changing the hex values in the text fields. You can use a website such as Kuler (https://color.adobe.com/) to find the right colors and hex values.
